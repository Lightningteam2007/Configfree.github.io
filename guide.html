راهنمای استفاده از کانفیگ‌ها

آموزش قدم‌به‌قدم نصب و استفاده از v2rayNG و کانفیگ‌های ما

دانلود v2rayNG

برای استفاده از کانفیگ‌های ما، باید اپلیکیشن v2rayNG رو نصب کنید. لینک‌های دانلود رسمی:

  • اندروید (GitHub)
  • اندروید (Google Play)
  • ویندوز (V2Ray Core)
  • مک (V2Ray Core)
  • iOS (از کلاینت‌های Shadowsocks یا V2Ray)

آموزش استفاده از کانفیگ‌ها

  1. وارد صفحه اصلی بشید و یه کانفیگ V2Ray یا VMess کپی کنید.
  2. اپلیکیشن v2rayNG رو باز کنید.
  3. به بخش تنظیمات برید و روی «+» بزنید.
  4. گزینه «Import config from clipboard» رو انتخاب کنید.
  5. کانفیگ کپی‌شده رو وارد کنید و ذخیره کنید.
  6. سرور رو انتخاب کنید و روی دکمه اتصال بزنید.
  7. از اینترنت آزاد لذت ببرید!

نکات پیشرفته

  • برای افزایش امنیت، از پروتکل‌های TLS یا WebSocket استفاده کنید.
  • اگه سرور کار نکرد، یه کانفیگ جدید تست کنید.
  • برای سرعت بهتر، سرورهای نزدیک به موقعیت جغرافیایی خودتون رو انتخاب کنید.
